# Brand Guide

## Цвета

| Название | HEX | Использование |
|----------|-----|--------------|
| Primary Orange | `#EF7D00` | Кнопки, акценты, активные состояния |
| Blue | `#2F7DC1` | Логотип "one", синие элементы |
| White | `#FFFFFF` | Фон, текст на цветных фонах |
| Text Primary | `#1A1A1A` | Основной текст |
| Text Secondary | `#6B6B6B` | Подписи, подзаголовки |
| Border | `#EFEFEF` | Границы карточек |

## Типографика

| Роль | Шрифт | Применение |
|------|-------|-----------|
| Заголовки | Gilroy | Все заголовки, навигация, кнопки |
| Текст | Cera Pro | Основной текст, описания |

## Логотип

**onebilim** — "one" в цвете `#2F7DC1`, "bilim" в цвете `#EF7D00`. Gilroy Bold, строчные буквы.

:::warning Запрещено
Не деформировать, не менять цвета, не добавлять эффекты.
:::
