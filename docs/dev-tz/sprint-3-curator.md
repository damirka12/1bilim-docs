# Спринт 3 — Роль Куратора

## Цель спринта
Реализовать полный функционал роли куратора: дашборд, проверка ДЗ, real-time чат, Live уроки с Google Meet, attendance-трекер.

---

## 3.1 Дашборд куратора

- [ ] Виджет «Мои ученики» — кол-во активных
- [ ] Виджет «Требуют внимания» — студенты без активности 3+ дней
- [ ] Виджет «ДЗ на проверку» — кол-во ожидающих
- [ ] Виджет «Непрочитанные сообщения» — кол-во чатов с непрочитанными
- [ ] Список марафонов куратора с прогрессом студентов

---

## 3.2 Проверка домашних заданий

- [ ] Список ДЗ только своих учеников
- [ ] Фильтры: ожидает / на доработке / все
- [ ] Просмотр содержимого ДЗ — текст, файлы, аудио
- [ ] Действия: Принять / На доработку / Отклонить
- [ ] Текстовый комментарий к ДЗ
- [ ] Голосовой комментарий куратора (запись аудио)
- [ ] Бонусные баллы при принятии ДЗ
- [ ] Версионирование — история всех попыток студента

---

## 3.3 Чат куратор ↔ студент

- [ ] WebSocket real-time соединение
- [ ] Список чатов с непрочитанными счётчиками
- [ ] Отправка текстовых сообщений
- [ ] Отправка изображений
- [ ] Отправка файлов
- [ ] Отправка голосовых сообщений (с waveform)
- [ ] Статусы: отправлено ✓ / прочитано ✓✓
- [ ] Индикатор «печатает...»
- [ ] Ответ на сообщение (reply)
- [ ] Пагинация истории (cursor-based)
- [ ] Push-уведомление при новом сообщении
- [ ] Email если получатель неактивен 24+ часов

---

## 3.4 Live уроки + Attendance

### Создание Live уроков
- [ ] Форма создания: название, тема, группа, дата, время, длительность
- [ ] Интеграция Google Calendar API → автогенерация Meet-ссылки
- [ ] Автоматическая отправка приглашений всем студентам группы
- [ ] Bulk создание — несколько уроков на неделю/месяц вперёд за один раз
- [ ] Редактирование и отмена урока

### Attendance — код присутствия (основной способ)
- [ ] Кнопка «Начать отметку» во время урока
- [ ] Генерация 4-значного кода (случайный)
- [ ] Куратор задаёт время действия кода
- [ ] Push-уведомление студентам с кодом
- [ ] Студент вводит код → статус «Присутствовал»
- [ ] По истечении TTL → автоматически «Отсутствовал»
- [ ] Кнопка «Остановить отметку» (деактивировать код досрочно)

### Attendance — ручная отметка (резервный способ)
- [ ] Список студентов урока
- [ ] Статусы: Присутствовал / Отсутствовал / Опоздал / Уважительная причина
- [ ] Комментарий к статусу
- [ ] Доступно до и после урока (в любое время)

### Аналитика посещаемости
- [ ] История всех уроков по каждому студенту
- [ ] Процент посещаемости студента
- [ ] Сводная таблица по каждому уроку (кто был / не был)
- [ ] Экспорт в Excel

---

## 3.5 Расписание куратора

- [ ] Внутренний экран приложения (не Google Calendar)
- [ ] Все Live уроки куратора по всем группам
- [ ] Переключение: месяц / неделя / день
- [ ] Создание события-объявления для студентов прямо из календаря
- [ ] Цветовая маркировка по типу события
