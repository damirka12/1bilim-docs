# Спринт 4 — Роль Студента

## Цель спринта
Реализовать полный функционал роли студента: прохождение уроков, ДЗ, CEFR, сторисы, словарь, рейтинг, календарь, профиль.

---

## 4.1 Прохождение уроков

- [ ] Главная страница — карусель сторисов, блок «Продолжить», карточки марафонов с прогрессом
- [ ] Список модулей (аккордеон) → разделы → блоки
- [ ] Статусы уроков: пройден / доступен / заблокирован
- [ ] Рендеринг всех типов блоков (информационные + интерактивные)
- [ ] Видеоплеер с запоминанием позиции просмотра
- [ ] Heartbeat каждые 30 сек → обновление time_spent
- [ ] Автопроверка интерактивных блоков (тесты, drag-drop, matching)
- [ ] Объяснение правильного ответа после теста
- [ ] Кнопка «Завершить урок» → начисление баллов
- [ ] Поиск урока по названию внутри марафона

---

## 4.2 Домашние задания (студент)

- [ ] Форма отправки ДЗ в конце раздела
- [ ] Прикрепление файлов (любые форматы, макс. 50MB)
- [ ] Запись аудио (макс. 3 минуты, автостоп)
- [ ] Текстовый ответ (для эссе-блоков)
- [ ] Статусы ДЗ: pending ⏳ / approved ✅ / revision 🔄 / rejected ❌
- [ ] Просмотр комментария куратора (текст + голосовой)
- [ ] Повторная отправка при статусе «на доработку»
- [ ] История всех попыток

---

## 4.3 CEFR модуль

- [ ] Отображение текущего уровня (A1–C2)
- [ ] Прогресс до следующего уровня в процентах
- [ ] Составляющие оценки: тесты, уроки, активность
- [ ] Прогноз: сколько уроков до следующего уровня
- [ ] График истории изменений уровня
- [ ] Отображение на карточке марафона

---

## 4.4 Сторисы

- [ ] Карусель сторисов в формате Instagram Stories
- [ ] Мультислайдовые сторисы (несколько слайдов)
- [ ] Изображения и видео до 60 сек
- [ ] Ссылка при нажатии
- [ ] Отметка «просмотрено» — не показывать повторно
- [ ] Глобальные и привязанные к марафону

---

## 4.5 Словарь (Vocabulary)

- [ ] Список добавленных слов с переводом, примером, аудио произношением
- [ ] Кнопка «Озвучить» (TTS)
- [ ] Добавление слова из блока урока (кнопка «Добавить в словарь»)
- [ ] Самостоятельное добавление слова вручную → автоперевод через OpenAI
- [ ] Статусы: new / learning / learned
- [ ] Добавить в избранное / убрать из избранного
- [ ] Фильтры: все / избранные / изученные / по уровню CEFR
- [ ] Удаление слова из словаря

---

## 4.6 Рейтинг и баллы

- [ ] Таблица лидеров марафона (топ-5 / топ-10 / все — по настройке Adminки)
- [ ] Место студента в рейтинге
- [ ] История начисления баллов
- [ ] Streak-трекер — серия активных дней
- [ ] Push-уведомление при достижении streak 7 дней

---

## 4.7 Календарь студента

- [ ] Внутренний экран приложения (студент не покидает платформу)
- [ ] Переключение: месяц / неделя / день
- [ ] Цветовая маркировка по типу события:
  - [ ] 🔵 Live уроки (создаются куратором автоматически)
  - [ ] 🟢 Speaking Rooms (при бронировании)
  - [ ] 🟠 События платформы (вебинары, дедлайны, экзамены)
  - [ ] ⚫ Личные события студента
- [ ] Карточка события при клике — детали + кнопка «Войти в Google Meet»
- [ ] Создание личных событий (название, дата, время, заметка, цвет)
- [ ] Редактирование и удаление личных событий
- [ ] Напоминания за 1 час и 15 минут до Live урока и Speaking Room (push)

---

## 4.8 Live уроки и Attendance (студент)

- [ ] Live уроки отображаются в Календаре
- [ ] Карточка урока: дата, время, куратор, тема
- [ ] Кнопка «Войти в Google Meet» в назначенное время
- [ ] Ввод 4-значного кода присутствия
- [ ] Push-уведомление с кодом во время урока
- [ ] Attendance-трекер: статистика посещений (только Premium / Individual)
- [ ] История по каждому уроку со статусом

---

## 4.9 Профиль студента

- [ ] Загрузка и обновление аватара
- [ ] Редактирование имени и фамилии
- [ ] Просмотр email (только чтение)
- [ ] Смена пароля
- [ ] Выбор языка интерфейса (KZ / RU / EN)
- [ ] Список купленных марафонов с датами доступа и прогрессом
- [ ] Кнопка «Продлить» для истёкших марафонов
- [ ] Настройки уведомлений — тогглы по группам + тихие часы
- [ ] Критические уведомления нельзя отключить
