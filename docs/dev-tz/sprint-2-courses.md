# Спринт 2 — Админ-панель

## Цель спринта
Реализовать полную админ-панель: аутентификация, управление марафонами, конструктор уроков, управление учениками и кураторами, конструктор тарифов.

---

## 2.1 Аутентификация и роли

- [ ] Регистрация (email + пароль)
- [ ] Вход → JWT access-токен (TTL 15 мин) + refresh-токен (TTL 7 дней)
- [ ] Обновление access-токена по refresh-токену
- [ ] Logout — отзыв refresh-токена
- [ ] Восстановление пароля по email (одноразовый токен TTL 1 час)
- [ ] Блокировка аккаунта после 5 неудачных попыток (на 15 мин)
- [ ] RBAC middleware: admin / curator / student
- [ ] Заголовки `X-User-ID`, `X-User-Role`, `X-Request-ID` через API Gateway

---

## 2.2 Марафоны и модули

- [ ] Список марафонов с поиском и фильтрами (статус / категория / язык / уровень)
- [ ] Создание марафона — все поля: название, обложка, фон, категория, язык, уровень CEFR
- [ ] Режимы прохождения: open / sequential / with_review
- [ ] Настройки рейтинга: show_rating, rating_scope, show_student_rank
- [ ] Публикация / архивирование марафона
- [ ] Дублирование марафона (копия без студентов и кураторов)
- [ ] CRUD модулей внутри марафона (drag-and-drop порядок)
- [ ] CRUD уроков внутри модуля (позиция, баллы за завершение, статус)
- [ ] CRUD разделов внутри урока (дедлайн, настройки ДЗ)

---

## 2.3 Конструктор блоков

### Информационные блоки
- [ ] Изображение (с подписью и alt)
- [ ] Карусель изображений
- [ ] GIF
- [ ] Видео (YouTube / Vimeo / загрузка файла + HLS)
- [ ] Аудио с плеером
- [ ] Статья / текст (rich-text)
- [ ] Заметка / Callout (info / warning / tip / danger)
- [ ] Ссылка (кнопка с внешним URL)
- [ ] Разделитель

### Интерактивные блоки (автопроверка)
- [ ] Тест без таймера (multiple choice, shuffle вариантов)
- [ ] Тест с таймером (countdown)
- [ ] Истина / Ложь / Не указано (True / False / Not Given)
- [ ] Перенести слово в пропуск (drag-and-drop)
- [ ] Выбрать форму слова (выпадающий список)
- [ ] Слово к изображению drag-and-drop
- [ ] Форма слова к изображению (select)
- [ ] Собрать предложение из слов (sentence builder)
- [ ] Отсортировать по колонкам
- [ ] Расставить текст по порядку
- [ ] Составить слово из букв (анаграмма)
- [ ] Сопоставить слова (matching)

### Блоки с проверкой куратором
- [ ] Ввести слово в пропуск (текстовый ввод)
- [ ] Слово к изображению (текстовый ввод)
- [ ] Запись аудио (голосовой ответ, макс. 3 мин)
- [ ] Сочинение (эссе, min/max слов)

### Блок «Слово» (Vocabulary)
- [ ] Поля: слово, перевод KZ/RU, объяснение, пример + перевод примера
- [ ] TTS аудио произношения
- [ ] Кнопка «Добавить в словарь» для студента

### Drag-and-drop конструктор
- [ ] Изменение порядка блоков через drag-and-drop
- [ ] Скрытие/показ блока (is_visible)
- [ ] Настройка баллов за каждый блок

---

## 2.4 Управление учениками и кураторами

- [ ] Список учеников с поиском и фильтрами (марафон / куратор / статус / активность)
- [ ] Создание ученика вручную
- [ ] Импорт учеников из CSV
- [ ] Карточка ученика — все поля профиля
- [ ] Статусы: active / deactivated / blocked / deleted
- [ ] Просмотр credentials (логин + пароль)
- [ ] Отправка письма-приглашения (логин + пароль)
- [ ] Зачисление на марафон (выбор тарифа, период доступа)
- [ ] Назначение / смена куратора
- [ ] Продление доступа к марафону
- [ ] Список кураторов — добавление в марафон, лимит учеников
- [ ] Распределение учеников между кураторами
- [ ] Статистика куратора: кол-во учеников, проверенных ДЗ, среднее время ответа

---

## 2.5 Конструктор тарифов

- [ ] Создание тарифа — название, описание, цена, валюта, длительность
- [ ] Конструктор состава тарифа (галочки: уроки, ДЗ, чат, Live, Speaking, Attendance, CEFR, Vocabulary)
- [ ] Реальная скидка (фактическое снижение цены)
- [ ] Фантомная скидка (зачёркнутая цена)
- [ ] Период действия скидки
- [ ] Активация / деактивация / архивирование тарифа
- [ ] Порядок отображения тарифов на странице оплаты
- [ ] Бейдж «Популярный» / кастомный текст
- [ ] Публичная страница курса — конструктор (обложка, описание, программа, тарифы)
- [ ] Порядок курсов на главной публичной странице

- [ ] Проверка ДЗ: принять / на доработку / отклонить
