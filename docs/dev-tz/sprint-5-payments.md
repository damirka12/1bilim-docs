# Спринт 5 — Оплата + Уведомления + i18n

## Цель спринта
Реализовать полный платёжный флоу (Kaspi + TipTopPay), систему уведомлений (push + email) и интернационализацию (KZ / RU / EN).

---

## 5.1 Оплата — Kaspi.kz

- [ ] Инициация платежа → получение payment_url от Kaspi
- [ ] Редирект студента на страницу оплаты Kaspi
- [ ] Обработка webhook от Kaspi (проверка HMAC подписи)
- [ ] Idempotency check — защита от дублей при повторном webhook
- [ ] Polling статуса как fallback (если webhook не пришёл за 5 мин)
- [ ] Автоматическое открытие доступа после успешной оплаты

---

## 5.2 Оплата — TipTopPay (Visa / Mastercard)

- [ ] Инициация платежа через TipTopPay
- [ ] 3D Secure (обрабатывается на стороне TipTopPay)
- [ ] Webhook обработка (проверка подписи)
- [ ] Возвраты через TipTopPay API
- [ ] Ручное предоставление доступа администратором

---

## 5.3 Флоу покупки (студент)

- [ ] Публичная страница → выбор курса и тарифа
- [ ] Регистрация / вход (если не авторизован)
- [ ] Страница подтверждения тарифа + итоговая сумма
- [ ] Выбор способа оплаты: Kaspi / банковская карта
- [ ] Страница успешной оплаты → редирект в ЛК
- [ ] Страница ошибки оплаты → вернуться к выбору тарифа
- [ ] История платежей в профиле студента

---

## 5.4 UTM-атрибуция

- [ ] Сохранение UTM-параметров при первом визите (utm_source, utm_medium, utm_campaign)
- [ ] Привязка UTM к каждой транзакции
- [ ] Отчёт в Adminке: продажи и выручка по источникам

---

## 5.5 Уведомления — Push (FCM / APNs)

- [ ] ДЗ проверено (принято / на доработку / отклонено) — критическое
- [ ] Новый урок открылся
- [ ] Новое сообщение в чате
- [ ] Напоминание о Live уроке (за 1 час и 15 минут)
- [ ] Attendance code — введи код сейчас — критическое
- [ ] Напоминание о Speaking Room (за 1 час и 15 минут)
- [ ] Доступ истекает через 3 дня / 1 день / сегодня — критическое
- [ ] Оплата прошла успешно — критическое
- [ ] Streak 7 дней достигнут
- [ ] Новый ученик назначен (куратору)
- [ ] Новое ДЗ на проверку (куратору)
- [ ] Ученик неактивен 3+ дней (куратору)
- [ ] Обработка невалидных FCM токенов (деактивация)
- [ ] Тихие часы — задержка push до окончания

---

## 5.6 Уведомления — Email (SendGrid / AWS SES)

- [ ] Welcome email при регистрации
- [ ] Письмо-приглашение студенту (логин + временный пароль)
- [ ] Сброс пароля
- [ ] ДЗ проверено (если неактивен 24+ часов)
- [ ] Подтверждение оплаты
- [ ] Истечение доступа (за 3 дня и в день X)
- [ ] Напоминание о Live уроке (за 1 час)
- [ ] Напоминание о Speaking Room (за 1 час)
- [ ] Шаблоны на трёх языках (KZ / RU / EN)
- [ ] Email отправляется только если пользователь неактивен 24+ часов (кроме критических)

---

## 5.7 Настройки уведомлений (студент)

- [ ] Экран настроек в профиле
- [ ] Тогглы по группам: уроки / чат / speaking rooms / оплата
- [ ] Тихие часы — включить, задать время начала и конца
- [ ] Критические уведомления нельзя отключить

---

## 5.8 i18n — три языка (KZ / RU / EN)

- [ ] Все экраны переведены на KZ / RU / EN
- [ ] Переключатель языка в профиле студента
- [ ] Язык писем определяется по `users.lang`
- [ ] Язык push-уведомлений определяется по `users.lang`
