# Оплата и тарифы

:::info
Все тарифы, цены и логика доступа управляются в Админке без участия разработчиков.
:::

## Способы оплаты

- **Kaspi.kz** — оплата картами Kaspi
- **TipTopPay** — банковские карты Visa / Mastercard

## UTM-аналитика

При каждой покупке фиксируется источник трафика: `utm_source`, `utm_medium`, `utm_campaign`.

Примеры: Instagram 1Bilim, Instagram Miss Dana, Google Ads.

## Флоу покупки

1. Главная → карточки курсов
2. Страница курса → выбор тарифа
3. Регистрация / Вход
4. Подтверждение тарифа + сумма
5. Оплата (Kaspi или банковская карта)
6. Доступ открыт → редирект в личный кабинет
