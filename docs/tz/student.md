# Роль Оқушы (Студент)

## Авторизация

- Email + пароль
- Восстановление пароля по email
- Получение приглашения с данными для входа

## Главная страница

- Карусель сторисов (Instagram Stories формат)
- Блок «Продолжить»: последний незавершённый урок
- Карточки активных марафонов с прогресс-баром
- Рейтинг (если включён Admin'ом)

## Прохождение уроков

- Список модулей (аккордеон) → разделы → блоки
- Статусы: пройден ✅ / доступен ▶ / заблокирован 🔒
- Видеоплеер с запоминанием позиции
- Интерактивные блоки: тесты, drag-and-drop, сопоставление
- Кнопка «Завершить урок» — начисление баллов

## Үй Тапсырмасы (ҮТ)

| Статус | Отображение | Действие |
|--------|-------------|----------|
| pending | На проверке ⏳ | Ждать |
| approved | Принято ✅ | Просмотреть комментарий |
| revision | Нужны исправления 🔄 | Исправить и отправить заново |
| rejected | Отклонено ❌ | Связаться с куратором |

**Ограничения файлов:**
- Максимальный размер: 50MB
- Аудио-запись: максимум 3 минуты
- Количество файлов: не ограничено

## CEFR Анализ уровня

- Автоматический расчёт уровня по шкале A1–C2
- Прогресс до следующего уровня в процентах
- Прогноз: сколько уроков осталось

## Speaking Rooms

Доступно только для тарифов **Premium** и **Individual**.

### Комнаты со студентами (Google Meet)
- Создание комнаты: тема, дата, длительность, макс. участников, уровень CEFR
- Бронирование с фильтрами по дате и уровню
- Отмена брони в любое время
- Оценка сессии (1–5 звёзд + комментарий)

### AI-бот (Speaking + Writing)
- **AI Speaking Trainer** — разговорная практика с AI-собеседником
- **AI Writing Trainer** — проверка эссе с обратной связью
- OpenAI GPT + Whisper STT + TTS
- Запускается мгновенно, без бронирования

## Словарь (Vocabulary)

- Слова из уроков + самостоятельное добавление
- Автоперевод через OpenAI при добавлении
- Фильтры: все / избранные / изученные / по уровню CEFR
- Кнопка озвучивания (TTS)

## Система баллов и рейтинг

| Действие | Баллы |
|----------|-------|
| Завершил урок | 10 |
| ҮТ принято куратором | 20 |
| Тест пройден | 5–15 |
| Был на Live уроке | 15 |
| Streak: 7 дней подряд | 50 |

| Тариф | Цена (KZT) | Что входит |
|-------|-----------|------------|
| Standard | 75 000 | Уроки, ҮТ, чат, CEFR, словарь |
| Premium | 150 000 | Standard + Live уроки, Speaking Rooms, Attendance |
