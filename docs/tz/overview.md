# Обзор проекта

**onebilim** — интерактивная платформа для онлайн-обучения языкам. Позволяет автоматизировать проведение уроков, создание учебного контента, отслеживание прогресса, организацию проверки үй тапсырмасы и общение между участниками.

## Три курса

| Курс | Описание |
|------|----------|
| **IELTS** | Подготовка к международному экзамену IELTS |
| **Pre-Upper** | Уровень Pre-Upper Intermediate |
| **Beginner** | Для начинающих с нуля |

## Роли пользователей

| Роль | Описание | Ключевые возможности |
|------|----------|----------------------|
| Admin | Полный доступ к системе | Марафоны, пользователи, контент, аналитика, оплата |
| Куратор | Куратор учеников | Свои ученики, чат, проверка ҮТ, расписание |
| Оқушы | Ученик | Уроки, ҮТ, чат с куратором, словарь, speaking rooms |

## Технологический стек

| Слой | Технологии |
|------|-----------|
| Frontend | React 18+, TypeScript, Tailwind CSS, TanStack Query, Zustand |
| Backend | Go 1.21+, Gin Framework, GORM, JWT Auth, WebSocket |
| База данных | PostgreSQL 15+, Redis |
| Хранилище файлов | AWS S3 / MinIO |
| Инфраструктура | Docker, GitHub Actions, CloudFlare CDN |
| AI/ML | OpenAI API (speaking rooms, CEFR-анализ) |
| Видеозвонки | Google Meet через Google Calendar API |
| Платежи | Kaspi.kz, TipTopPay |
