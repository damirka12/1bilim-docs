# Система уведомлений

Каналы: **Push** (FCM / APNs) и **Email**.

## Таблица событий

| Событие | Push | Email | Кому | Отключить? |
|---------|------|-------|------|-----------|
| ҮТ проверено | ✅ | ✅ | Оқушы | ❌ |
| Новый урок открыт | ✅ | ❌ | Оқушы | ✅ |
| Новое сообщение в чате | ✅ | ✅* | Оба | ❌ |
| Напоминание о Live уроке | ✅ | ✅ | Оқушы | ✅ |
| Attendance code | ✅ | ❌ | Оқушы | ❌ |
| Напоминание о Speaking Room | ✅ | ✅ | Оқушы | ✅ |
| Доступ истекает (3 дня) | ✅ | ✅ | Оқушы | ❌ |
| Доступ истёк | ✅ | ✅ | Оқушы | ❌ |
| Оплата прошла | ✅ | ✅ | Оқушы | ❌ |
| Новый ученик назначен | ✅ | ❌ | Куратор | ✅ |
| Новое ҮТ на проверку | ✅ | ❌ | Куратор | ✅ |
| Ученик не активен 3+ дней | ✅ | ❌ | Куратор | ✅ |

*Email отправляется только если пользователь не открывал приложение 24+ часов.

## Настройки (Оқушы)

Отдельный экран в профиле. Тогглы по группам:
- Уроки
- Чат
- Speaking Rooms
- Оплата

:::warning Критические уведомления
Нельзя отключить: ҮТ проверено, оплата, истечение доступа, attendance code.
:::
