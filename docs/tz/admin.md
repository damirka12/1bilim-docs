# Админ-панель

## Марафоны (Курсы)

Марафон — основная учебная единица платформы. Три курса: **IELTS**, **Pre-Upper**, **Beginner**.

### Поля марафона

| Поле | Тип | Описание |
|------|-----|----------|
| title | string | Название марафона |
| cover_url | string | Обложка |
| category | enum | языки / бизнес / подготовка к экзаменам |
| language | enum | KZ / RU / EN |
| level_start | enum | Начальный уровень CEFR (A1–C2) |
| level_end | enum | Финальный уровень CEFR (A1–C2) |
| access_mode | enum | open / sequential / with_review |
| show_rating | boolean | Показывать рейтинг ученикам |
| status | enum | draft / published / archived |

### Режимы прохождения

| Режим | Код | Описание |
|-------|-----|----------|
| Открытый | open | Все уроки доступны сразу |
| Постепенный | sequential | Следующий урок открывается после завершения предыдущего |
| Проверка учителем | with_review | Следующий урок открывается после принятия ҮТ куратором |

## Конструктор уроков

Drag-and-drop редактор. Урок → Разделы → Блоки контента.

### Типы блоков контента

| Категория | Тип блока | Описание |
|-----------|-----------|----------|
| Изображение | Картинка | Одиночное изображение с подписью |
| Изображение | Карусель | Несколько изображений в слайдере |
| Аудио и видео | Видео | YouTube / Vimeo или загрузка |
| Аудио и видео | Запись аудио | Ученик записывает голос |
| Слова и пропуски | Drag-and-drop | Перенести слово в пропуск |
| Слова и пропуски | Ввод текста | Ввод слова в пропуск |
| Тесты | Без таймера | Multiple choice |
| Тесты | С таймером | Multiple choice с обратным отсчётом |
| Порядок | Предложение из слов | Собрать предложение |
| Порядок | Сопоставить слова | Matching |
| Работа с текстом | Статья | Rich-text для чтения |
| Работа с текстом | Сочинение | Развёрнутый ответ — проверка куратором |
| Прочее | Набор слов | Карточки словаря урока |

## Управление учениками

- Поиск и фильтр (по марафону, куратору, статусу, активности)
- Импорт CSV / ручное добавление
- Назначить/сменить куратора
- Продлить доступ к марафону

### Статусы ученика

| Статус | Описание |
|--------|----------|
| active | Активный |
| deactivated | Деактивирован |
| blocked | Заблокирован |
| deleted | Удалён |

## Кураторы

- Добавление / удаление куратора в марафон
- Распределение учеников между кураторами
- Лимит учеников на куратора
- Статистика: кол-во учеников, проверенных ҮТ, среднее время ответа

## Сторисы

- Создание объявлений в формате Instagram Stories
- Медиа: изображение или видео до 60 сек
- Привязка к марафону или глобально
- Статистика просмотров

## Статистика

- Общий дашборд: активные пользователи, завершённые уроки, выручка
- Аналитика по марафонам: прогресс, завершаемость
- Аналитика по ҮТ: скорость проверки, процент принятых
- Эффективность кураторов
- Скачать данные в Excel
