# Роль Куратора

## Дашборд

- Мои ученики: кол-во активных, требующих внимания (3+ дней без активности)
- ҮТ на проверку: кол-во ожидающих
- Непрочитанные сообщения в чатах

## Проверка ҮТ

- Список ҮТ только своих учеников
- Фильтры: ожидает / на доработке / все
- Действия: **Принять** / **На доработку** (с комментарием) / **Отклонить**
- Обратная связь: текст, голосовое сообщение, бонусные баллы

## Live уроки

Куратор создаёт Live уроки для своих групп через Google Meet.

- Название и тема урока
- Группа, дата, время, длительность
- Платформа автоматически создаёт событие в Google Calendar
- Все студенты получают приглашение автоматически
- Bulk создание: несколько уроков на неделю/месяц вперёд

## Посещаемость (Attendance)

### Основной способ — Attendance code

1. Куратор нажимает «Начать отметку» во время урока
2. Генерируется код из 4 цифр
3. Студенты получают push-уведомление → вводят код
4. Ввёл в срок → Присутствовал. Не ввёл → Отсутствовал

### Резервный способ — Ручная отметка

Куратор открывает список студентов и ставит статус вручную.

### Статусы посещаемости

| Статус | Как ставится |
|--------|-------------|
| Присутствовал | Авто (код) / Ручной |
| Отсутствовал | Авто (истёк код) / Ручной |
| Опоздал | Только ручной |
| Уважительная причина | Только ручной + комментарий |

## Чат

- Личные чаты с каждым своим учеником (1:1)
- Типы сообщений: текст, изображение, файл, голосовое
- Индикатор: отправлено ✓ / прочитано ✓✓
- Push-уведомления при неактивности 24+ ч

## Расписание

- Внутренний экран — не Google Calendar
- Все Live уроки куратора по всем группам
- Создание событий-объявлений для студентов
